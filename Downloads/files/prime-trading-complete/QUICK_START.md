# ⚡ QUICK START GUIDE

## 5-Minute Setup

### 1. Extract & Install
```bash
cd prime-trading-complete
npm install
```

### 2. Configure
```bash
cp .env.local.example .env.local
# Edit .env.local with Stripe keys
```

### 3. Run
```bash
npm run dev
# Visit http://localhost:3000
```

### 4. Deploy
```bash
vercel
# Follow prompts
```

### 5. Done! 🎉

Your site is now live on Vercel!
