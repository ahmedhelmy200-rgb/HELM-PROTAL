## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- Git
- Stripe account

### Installation (30 minutes)

1. **Extract ZIP and navigate to folder**
   ```bash
   cd prime-trading-complete
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.local.example .env.local
   # Edit .env.local and add your Stripe keys
   ```

4. **Test locally**
   ```bash
   npm run dev
   # Open http://localhost:3000
   ```

5. **Deploy to Vercel**
   ```bash
   npm i -g vercel
   vercel
   ```

6. **Push to GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/prime-trading.git
   git push -u origin main
   ```
