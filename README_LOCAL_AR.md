# تشغيل النظام محليًا

1. ثبّت Node.js 22 أو أحدث.
2. انسخ `.env.example` إلى `.env.local` وضع قيم Base44 إذا كانت مطلوبة.
3. شغّل:

```powershell
./run-local.ps1
```

أو يدويًا:

```powershell
npm install
npm run dev
```

## ملاحظات
- المشروع الآن مُجمع كملفات كاملة من المحادثة + ملف `New folder.zip`.
- الواجهة المحلية يمكن تشغيلها مباشرة بعد تثبيت الاعتماديات.
- وظائف Base44 (الدخول، البيانات، الرفع، OCR) تحتاج قيم التطبيق الصحيحة حتى تعمل بالكامل.
